# Online Code Snippet Manager

## Description
Online Code Snippet Manager is a web application that allows users to save, organize, search, edit, and manage code snippets efficiently.

## Features
- Add new code snippets
- Edit existing snippets
- Delete snippets
- Search snippets
- Filter by programming language
- Filter by tags
- Mark snippets as favorites
- Copy code to clipboard
- Local Storage support
- Syntax Highlighting

## Technologies Used
- HTML
- CSS
- JavaScript
- Highlight.js

## How to Run
1. Download or clone the repository.
2. Open `online_code_snippet_manager.html` in any modern web browser.
3. Start using the application.
