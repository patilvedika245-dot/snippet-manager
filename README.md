# Online Code Snippet Manager

## Description
Online Code Snippet Manager is a web application that allows users to save, organize, search, edit, and manage code snippets efficiently.

## Features
- Add new code snippets
- Edit existing snippets
- Delete snippets
- Search snippets
- Filter by programming language
- Filter by tags
- Mark snippets as favorites
- Copy code to clipboard
- Local Storage support
- Syntax Highlighting

## Technologies Used
- HTML
- CSS
- JavaScript
- Highlight.js

## How to Use

1. Open the application in your browser.
2. Click **New Snippet** to add code.
3. Enter the title, language, tags, and code.
4. Save the snippet.
5. Use the search bar to find snippets.
6. Mark important snippets as favorites.
